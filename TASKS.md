## Definition of Done

- Сотрудник входит в админку по логину и паролю.
- Сотрудник может добавить ребёнка с ФИО, ИИН, телефоном и необязательным сроком начала занятий.
- Активная очередь работает по FIFO и автоматически пересчитывается после зачисления или удаления.
- Сотрудник может искать и фильтровать детей, редактировать данные, удалять запись в архив и повторно отправлять ссылку.
- Статусы проходят этапы из PRD, поддерживают подтверждённый откат ошибки и корректно влияют на активную очередь.
- Родитель открывает персональную ссылку без входа и видит только данные своего ребёнка.
- Родитель может найти статус по ИИН, при этом действует ограничение попыток.
- Система готовит тексты сообщений со ссылкой при постановке в очередь, смене этапа и агрегированном продвижении очереди.
- Сотрудник может скопировать персональную ссылку или готовый текст сообщения для ручной отправки родителю.
- Действия сотрудников фиксируются в журнале аудита.
- Интерфейс на русском языке, соответствует палитре PRD, адаптирован для мобильных устройств и проходит базовую проверку контрастности.
- В MVP не попали функции, явно указанные в PRD как вне области первой версии.
